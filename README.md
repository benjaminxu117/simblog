# simblog
self made simple blog, with read doc, subscribe, like, user create. using java spring boot

# Java Project

这是一个基本的Java项目，使用Maven作为构建工具。

## 环境要求

- Java 11 或更高版本
- Maven 3.6 或更高版本

## 如何运行

1. 编译项目：
```bash
mvn compile
```

2. 运行测试：
```bash
mvn test
```

3. 运行应用程序：
```bash
mvn exec:java -Dexec.mainClass="com.example.App"
```

## 项目结构

```
src/
├── main/java/        # 主要源代码
└── test/java/        # 测试代码
```
